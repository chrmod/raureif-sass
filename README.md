# raureif-sass

it is a [raureif](https://github.com/chrmod/raureif) addon that adds a SASS
compiler to raureif project.

## Installation

`npm install --save-dev raureif-sass`

## Usage

Files with `.scss` extension in `src/styles` folder will be compiled to `.css`.

Files that should not be compiled to individual `.css` should start with `_` symbol.
